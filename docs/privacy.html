AlgoNotion 개인정보처리방침

최종 업데이트: 2026년 3월 28일

AlgoNotion(이하 "확장 프로그램")은 백준(acmicpc.net)과 SWEA(swexpertacademy.com)에서 맞은 알고리즘 문제 제출을 감지하여 사용자가 지정한 Notion 데이터베이스에 자동으로 업로드하는 Chrome 확장 프로그램입니다.

수집하는 데이터

  • Notion API 토큰: 사용자가 옵션 페이지에서 직접 입력하며, Notion 데이터베이스에 접근하기 위해 사용됩니다.
  • Notion 데이터베이스 ID: 사용자가 옵션 페이지에서 직접 입력하며, 업로드 대상 데이터베이스를 식별하기 위해 사용됩니다.
  • 제출 소스코드 및 문제 메타데이터: 맞은 제출의 소스코드, 문제 번호, 제목, 난이도가 Notion에 업로드하는 용도로만 사용됩니다.

데이터 저장 방식

  • Notion API 토큰과 데이터베이스 ID는 chrome.storage.local에 사용자 기기 내에만 저장됩니다.
  • 업로드된 제출 ID 목록(중복 방지용)도 동일하게 로컬에만 저장됩니다.
  • 수집된 데이터는 외부 서버에 영구 저장되지 않습니다.

데이터 전송

소스코드 및 문제 메타데이터는 확장 프로그램의 중계 서버를 통해 사용자 본인의 Notion 데이터베이스로 전송됩니다. 중계 서버는 데이터를 저장하거나 기록하지 않으며, Notion API 호출을 위한 단순 중계 역할만 수행합니다.

제3자 데이터 공유

수집된 데이터는 광고, 분석, 판매 등 어떠한 목적으로도 제3자와 공유되지 않습니다.

권한 사용 목적

  • storage: Notion 토큰, 데이터베이스 ID, 중복 방지용 제출 ID를 로컬에 저장
  • activeTab / scripting: 백준·SWEA 페이지에서 제출 결과 감지 및 업로드 버튼 삽입
  • declarativeNetRequest: 백준 소스코드 다운로드에 필요한 요청 헤더 수정
  • tabs: 현재 탭이 지원되는 사이트인지 확인
